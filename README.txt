This plugin will delete post image with post.

== Description ==

Are you want to delete featured image with post this plugin will help you to do this.

Just install the plugin and enjoy, No need any settings.

If post type is product then it will delete WooCommerce gallery images too.

If you need more feature on this plugin, Please send me a mail sourovfci@gmail.com i will add your feature.

This plugin is also in github
https://github.com/sourovroy/delete-post-swrs

== Installation ==

1. Upload `delete-image-with-post` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. No need to do any settings
